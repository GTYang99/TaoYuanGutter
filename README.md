# 桃園側溝 - Android GutterForm 應用程式

這是 Android Kotlin 專案，用於管理 GutterForm 資料，包含擷取基本資訊、處理照片，以及整合地圖進行位置標記。

## 主要功能

-   **側溝基本資訊表單 (Gutter Basic Information Form):** 收集側溝的基本詳細資訊。
-   **照片管理 (Photo Management):** 支援最多三張照片的拍攝與顯示。
-   **地圖整合 (Map Integration):** 允許在地圖上標記位置。
-   **編輯模式 (Edit Mode):** 優化用於編輯現有記錄，進行特定 UI 調整，例如隱藏側溝 ID 欄位及預先填入測量 ID。
-   **草稿管理 (Pending Drafts):** 管理並恢復側溝表單的已儲存草稿。
-   **點位管理 (Waypoint Management):** 包含點位的滑動刪除功能。

## 目前狀態與已解決問題

*   **提交流程:** `fabSubmit` 按鈕的點擊流程已修正，確保在提交前已完成基本資訊和照片的填寫。
*   **驗證邏輯:** 已實作基本資訊和照片的完整驗證邏輯。
*   **照片處理:** `FragmentGutterPhotos` 完全支援照片選擇與顯示，現已使用 `Uri` 以獲得更好的相容性。
*   **編輯模式資料顯示:** 已解決編輯模式下點位資料的顯示問題，確保從檢視畫面能正確載入。
*   **表單行為:** 編輯模式下的表單行為已優化，包括隱藏「側溝 ID」欄位，並使用 API 資料自動填入「測量 ID」。
